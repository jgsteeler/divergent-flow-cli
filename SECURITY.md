# Security Policy

## Supported Versions

We generally support the latest released version of the CLI.

## Reporting a Vulnerability

If you discover a security vulnerability, please email us at [security@gibsonserviceco.com](mailto:security@gibsonserviceco.com) with details. We will acknowledge your report within 3 business days and work with you to resolve the issue swiftly.

Please do not file public issues for security problems.
